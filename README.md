# flow-example

This application requires on JDK 17 and Apache Maven.

# Setup

- [Apache Maven](https://maven.apache.org/index.html)
- [Azul Zulu JDK](https://www.azul.com/downloads/?version=java-17-lts&package=jdk)

# Maven build

```
mvn clean test
```

# Running the server locally

```
make run
```
